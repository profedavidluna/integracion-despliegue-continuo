# Módulo I: Integración y Despliegue Continuo con Contenedores

## Descripción General

Este módulo es el punto de partida del curso. Aquí construirás los cimientos que necesitas para entender cómo los contenedores transformaron la forma en que las empresas desarrollan, prueban y despliegan software. Al finalizar el módulo, serás capaz de contenerizar una aplicación real, orquestar un entorno multi-servicio con Docker Compose e identificar las prácticas que aplican los equipos de ingeniería de clase mundial.

---

## Objetivos de Aprendizaje

Al completar este módulo, el estudiante será capaz de:

- Explicar qué es un contenedor y diferenciarlo de una Máquina Virtual.
- Describir los mecanismos del kernel de Linux que hacen posibles los contenedores.
- Construir una imagen Docker optimizada para producción utilizando buenas prácticas.
- Definir y levantar un entorno multi-contenedor con Docker Compose.
- Ejecutar los comandos esenciales de Docker y Docker Compose con confianza.
- Relacionar los conceptos técnicos con casos de éxito empresarial reales.
